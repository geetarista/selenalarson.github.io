--- layout: post title: "Facebook Announces New APIs for Comment Replies" date: 2013-04-03 categories: tech --- The new Facebook API allows developers to build tools that make it easier for brands to monitor and respond to comment replies. This comes on the heels of last week’s announcement that Facebook will allow comment replies on pages and profiles with more than 10,000 followers.

Pages can choose to activate this new comment feature by going to their Admin Panel and selecting “Manage permissions.” Facebook tested the new comment feature on Pages like ESPN and the Huffington Post and received positive reviews. Although this feature is still optional, it will implemented to all pages and profiles with more than 10,000 followers in July.

The new API now supports different “views” of the comments on posts through the updated comments API and FQL comment table.

The documentation includes new details on how to use the features including reading comments, creating comment replies, and deleting comment replies. The Roadmap Facebook has posted lists detailed information on the API changes and outlines all modifications going into effect permanently on July 10.
